Конспект. Git, terminal.

PWD – команда показывает, где находимся (путь)
CD – change directory. Вариации: cd .../ (назад), cd C:/QA/ (перейти в путь), cd (в начало). 
Если ввести начало папки и нажать tab – название дополнится.
MKDIR – make directory (mkdir qa_1, также mkdir qa_1, qa_2, qa_3) – создать папку
LS -LA – вывести содержание папки. Можно LS просто без параметра (выведет папки в строку), 
можно ls -l – тогда выведет без скрытых файлов. LS -A выведет со скрытыми, но в строку.
TOUCH QQ.TXT – создать пустой файл в текущей папке.
CAT >> QQ.TXT открывает файл с редактором (закрыть ctrl+c). Той же командой можно дополнить файл.
CAT QQ.TXT – вывести содержание файла.
CAT > QQ.TXT – создать файл и открыть сразу редактор. CAT > QA_1/QQ.TXT (создать файл в папке).
HISTORY – история команд. Можно листать через «вверх» и «вниз» по командам. Выйти из режима -q.
 Текстовый редактор vim
VIM QQ.TXT – открывается.
Нажимаем I для режима редактирования. Ворошим.
Чтобы выйти нажимаем ESC, вводим :WQ, жмем ENTER. Можно без сохранения :q!.

MV QA_1/QQ.TXT QA_3/QQ.TXT – move что и куда. Перемещение файлов. Этой же командой можно переименовать, напр.: MV QQ.TXT Q99.TXT
Аналогично CP – copy, копирование.
Rm QQ.TXT – remove, удалить файл.
RM -R QA_1 – удалить папку. Можно воспользоваться RMDIR, если папка пустая.
